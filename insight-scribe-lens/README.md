# Insight Scribe Lens

AI Analytics Dashboard built by **Rakesh Reddy**.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone https://github.com/sairakesh-143/insight-scribe-lens.git

# Navigate to the project directory
cd insight-scribe-lens

# Install dependencies
npm install

# Start the development server
npm run dev
```

- TypeScript
- React
